1. Mengenal HTML, CSS, JavaScript & Tailwind

🧱 HTML (HyperText Markup Language)

HTML adalah bahasa dasar untuk buat halaman web. HTML berfungsi untuk membentuk struktur atau kerangka dari sebuah website. Dari HTML, kita bisa menampilkan berbagai elemen seperti teks, gambar, tabel, video, dan tautan. HTML ibarat tulang dan organ tubuh manusia yang membentuk struktur dasar sebelum dikasih warna atau gaya.

🎨 CSS (Cascading Style Sheets)

CSS digunakan untuk mengatur tampilan dan desain halaman web. Pake CSS, kita bisa kasih warna, mengatur ukuran huruf, menata posisi elemen, juga buat tampilan website lebih menarik dan rapi. Kalau HTML adalah kerangkanya, maka CSS adalah pakaian dan riasan yang mempercantik tampilan luar website.

⚙️ JavaScript

JavaScript adalah bahasa pemrograman yang menambahkan interaktivitas dan logika di halaman web. Dengan JavaScript, website bisa jadi lebih hidup — misalnya menampilkan pesan saat tombol diklik, membuat animasi, atau memvalidasi data input pengguna. JavaScript berfungsi seperti otak dan saraf pada manusia, yang membuat website bisa bereaksi terhadap tindakan pengguna.

🎥 Video Penjelasan HTML, CSS, dan JavaScript

💨 Tailwind CSS

Tailwind CSS adalah framework CSS modern yang memudahkan proses desain tampilan website. Dengan Tailwind, pengembang tidak perlu menulis kode CSS panjang karena sudah ada berbagai class siap pakai. Tailwind CSS membuat pembuatan desain lebih cepat, efisien, dan tetap terlihat profesional. Framework ini ibarat pakaian instan yang bisa langsung dipakai dan dikombinasikan sesuai kebutuhan.

2. Tata Cara Membuat Website Sederhana

  1. Buka VSCode dan buat folder baru.
  2. Buat file: index.html, style.css, dan script.js.
  3. Tulis struktur HTML dasar.
  4. Tambahkan style dengan CSS dan efek dengan JavaScript.
  5. Buka hasilnya di browser untuk melihat tampilan website!
